# 🍔 Food - Landing Page para Restaurante

![Preview do Projeto](./src/assets/preview.png) Um projeto de uma landing page moderna, responsiva e animada para um restaurante/delivery. Este site foi desenvolvido com foco em uma interface de usuário (UI) atraente, com micro-interações, layout adaptável para mobile, e um incrível slider interativo para o cardápio de bebidas especiais!

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5** (Semântica e estruturação)
* **CSS3** (Estilização, Flexbox, variáveis, Animações e Keyframes)
* **JavaScript** (Lógica de interatividade e manipulação do DOM)
* **jQuery** (Facilitação da sintaxe JS e controle de eventos)
* **ScrollReveal.js** (Animações de entrada ao rolar a página)
* **Font Awesome** (Ícones)
* **Google Fonts** (Fonte Poppins)

## ✨ Funcionalidades

* **Design 100% Responsivo:** O layout se adapta perfeitamente a celulares, tablets e desktops (Mobile First/Media Queries).
* **Animações de Scroll:** Elementos surgem suavemente na tela conforme o usuário desce a página.
* **Slider de Bebidas Especiais:** Uma seção interativa que apresenta bebidas com efeitos de transição de cores e animações de queda independentes para as latas e frutas.
* **Navegação Inteligente:** Menu fixo que detecta a seção atual e muda de estado visualmente durante o scroll.
* **Menu Mobile:** Menu estilo "hambúrguer" totalmente funcional para telas menores.
* **Links de Contato:** Botões interativos que redirecionam para WhatsApp e redes sociais.
